### 我在做的事

**🦞 Claworld Desktop · 让 AI agent 住进一间像素办公室**

本地运行的 AI agent 可视化面板，深度集成 OpenClaw。打开就能实时看到你的 AI 在忙什么——一只龙虾在像素工位上替你干活。Claworld 世界观的入口。

[![claworld-desktop](https://img.shields.io/github/stars/David0936/claworld-desktop?style=flat&label=Claworld%20Desktop)](https://github.com/David0936/claworld-desktop)

**📈 Claworld Monitor · 盯财经博主，第一时间抓股票线索**

自部署的 X(Twitter) 财经监控系统：多账号实时监控 → AI 翻译/中文标题/财经解读 → 推文秒筛 A 股 + 美股 → 飞书弹卡片。博主一发推，十几秒到一分钟内点名相关股票。

[![claworld-x-monitor](https://img.shields.io/github/stars/David0936/claworld-x-monitor?style=flat&label=Claworld%20Monitor)](https://github.com/David0936/claworld-x-monitor)

**🧠 人物思维 Skill · 把"怎么想"做成可运行的代码**

不是让 AI 模仿某个人说话，而是提炼他们怎么做判断。比如 CIC.skill——关娜西从云南姑娘到把"半永久"做成行业代名词的 6 个心智模型，结构化成可复用的思维框架。

### 我相信的几件事

**稀缺的不是 AI 生成的东西，是判断力。** 知道做什么、不做什么，这件事得靠人想清楚——剩下的交给 agent。

**蒸馏比堆叠有用。** 我不收集 prompt，我提炼认知：一个 Skill 应该装得下一个人怎么思考，而不只是怎么说话。

**做出来比说出来有用。** 与其讲 AI 能干嘛，不如让你看到龙虾在我的办公室里真的在干活。
